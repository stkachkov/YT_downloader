# yt-dlp Downloader GUI

Простое десктопное приложение с графическим интерфейсом для параллельной загрузки видео с помощью `yt-dlp`.


## Возможности

*   Простой интерфейс: вставьте ссылку и нажмите Enter.
*   Параллельная загрузка нескольких видео.
*   Отслеживание прогресса для каждой загрузки в реальном времени.
*   Отображение названий видео.
*   Использование вашей собственной конфигурации `yt-dlp` для путей сохранения и форматов.

---

## 1. Предварительные требования

Перед использованием этого приложения вам необходимо установить `yt-dlp` в вашей системе. Это приложение является лишь графической оболочкой для него.

➡️ **[Инструкции по установке yt-dlp](https://github.com/yt-dlp/yt-dlp#installation)**

После установки убедитесь, что команда `yt-dlp` доступна из вашего терминала.

## 2. Настройка yt-dlp (Рекомендуется)

Вы можете управлять настройками `yt-dlp` (например, куда сохранять видео и в каком формате) с помощью конфигурационного файла.

1.  В этом репозитории есть файл `yt-dlp.conf.example`. Он содержит рекомендуемые настройки для "красивых" имен файлов и выбора качества.
2.  Создайте папку `yt-dlp` в вашей директории с конфигурациями. Для Linux это `~/.config/`.
3.  Скопируйте `yt-dlp.conf.example` в `~/.config/yt-dlp/` и переименуйте его в `yt-dlp.conf`.
4.  Откройте `~/.config/yt-dlp/yt-dlp.conf` и измените настройки по своему усмотрению. Например, вы можете раскомментировать и изменить путь сохранения видео (`-P "~/Downloads"`).

## 3. Использование

Есть два способа использовать это приложение: запустить из исходного кода или собрать в standalone-файл.

### Запуск из исходного кода

Этот способ подходит для разработчиков или для тех, кто хочет внести изменения.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/stkachkov/YT_downloader.git
    cd YT_downloader
    ```

2.  **Создайте и активируйте виртуальное окружение:**
    ```bash
    python3 -m venv .venv
    source .venv/bin/activate
    ```

3.  **Установите зависимости:**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Запустите приложение:**
    ```bash
    python3 main.py
    ```

### Сборка standalone-приложения

Этот способ позволяет создать один исполняемый файл, который можно запускать где угодно без установки Python и зависимостей.

1.  **Выполните шаги 1-2 из предыдущего раздела** (клонирование и создание окружения).

2.  **Запустите скрипт сборки:**
    ```bash
    ./build.sh
    ```
    *Если вы получите ошибку "Permission denied", выполните `chmod +x build.sh` один раз.*

3.  **Найдите ваше приложение** в папке `dist/`. Готовый файл будет называться `main`. Вы можете скопировать его в любое удобное место и запускать двойным щелчком.
